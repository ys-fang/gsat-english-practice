## 🌐 部署選項

### 選項1：靜態網站部署 (推薦)

**適用平台：**
- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting

**步驟：**
1. 上傳整個資料夾到Git repository
2. 設定platform連接repository
3. 設定根目錄為專案根目錄
4. 部署完成！

### 選項2：本地部署

**直接開啟：**
```bash
# 直接用瀏覽器打開
open index.html
```

**或使用簡單HTTP伺服器：**
```bash
# Python 3
python -m http.server 8080

# Node.js
npx serve .

# PHP
php -S localhost:8080
```
